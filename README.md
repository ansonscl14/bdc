# Number System Converter

A Python program that converts numbers between Binary, Decimal, and Hexadecimal.

## Requirements

- Python 3.x

No external libraries are required.

## How to Run

1. Clone the repository or download the project.
2. Open a terminal in the project folder.
3. Run:

```bash
python convert.py
```

## Example

```
Welcome to Binary - Decimal - Hexadecimal Converter

Choose your input format:
1 - Binary
2 - Decimal
3 - Hexadecimal

Choice: 1

Enter a binary number: 10110110

Binary: 10110110
Decimal: 182
Hexadecimal: B6
```
